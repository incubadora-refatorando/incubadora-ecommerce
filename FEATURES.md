# � Features Principais

## 🔧 Backend

**O que o backend faz:**

### Autenticação

- Cadastro de usuários
- Login com token JWT
- Proteção de rotas

### Produtos

- Listar todos os produtos
- Ver detalhes de um produto
- Criar/editar produtos (admin)

### Carrinho

- Adicionar produtos
- Remover produtos
- Atualizar quantidade

### Pedidos

- Criar pedido (checkout)
- Ver histórico de compras
- Ver detalhes do pedido

---

## 🎨 Frontend

**As páginas que você vê:**

| Página          | O que tem                     |
| --------------- | ----------------------------- |
| Login           | Formulário de email e senha   |
| Cadastro        | Formulário para criar conta   |
| Produtos        | Lista de produtos com imagem  |
| Detalhes        | Foto grande + info do produto |
| Carrinho        | Itens que quer comprar        |
| Pedidos         | Histórico de compras          |
| Detalhes Pedido | Info completa de uma compra   |

---

## 🔄 Como Funciona

```
1. Pessoa faz cadastro
2. Pessoa faz login (recebe token)
3. Pessoa ve lista de produtos
4. Pessoa adiciona ao carrinho
5. Pessoa checkout → cria pedido
6. Pessoa vê pedido em "Meus Pedidos"
```
